QChat

QChat is a simple real-time chat website built with HTML, CSS, JavaScript, and Firebase. It allows users to sign in with their mobile number and chat in real time.

Features

- Mobile number login using Firebase Authentication
- OTP verification
- User profile with name and profile picture
- Real-time messaging using Firestore
- Responsive design for mobile and desktop
- Firebase Storage for profile images
- Secure cloud database

Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

Installation

1. Download or clone the project.
2. Create a Firebase project.
3. Enable Phone Authentication.
4. Create a Cloud Firestore database.
5. Enable Firebase Storage.
6. Copy your Firebase configuration into "firebase.js".
7. Open "index.html" in your browser or deploy the project to Firebase Hosting, Netlify, or Vercel.
